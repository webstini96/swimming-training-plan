Session 2 — polish and deploy
3-4 hrs
Fix session 1 issues. Improve UI — dark theme, animations, pool size counters, dice visuals. Deploy to Vercel — public URL, accessible from any device. Basic lobby flow, waiting room, session timer.

Milestone 1: send the URL to 5 people you trust. Watch them play. Note every moment of confusion or delight.

Streamer proof of concept
Week 2-3
Session 3 — stream overlay and spectator UX
3-4 hrs
Build a clean stream-ready spectator view — large pool size displays, animated chip flows, claim history ticker, player dice display formatted for screen capture. OBS-compatible overlay layer.
Deliverable: spectator view that looks good on a Twitch stream
Milestone 2: recruit one mid-tier crypto or gaming streamer. Run a live session on stream. Capture the clip.
Milestone 3: the clip exists. This is your proof of concept for every conversation that follows — operators, investors, press.
Phase 2 — go on-chain
3
Web3 migration — Locker on-chain
Week 3-4
Session 4 — smart contract
4-5 hrs
Write the pool mechanics smart contract in Solidity — commit to pool, distribute pari-mutuel, deduct rake, send to prize pot, settle on round end. Deploy to testnet (Sepolia). Test every scenario with fake ETH.
Deliverable: pool contract running on testnet, every payout verifiable on-chain
Session 5 — wallet connection
3-4 hrs
wagmi + ConnectKit for wallet connection. Replace virtual chip balances with wallet reads. Pool commits become contract calls. Prize pot distributions become on-chain transfers. Web2 pool logic abstraction pays off here — swap the layer, not the game.
Deliverable: real crypto flowing through Locker on testnet
Before mainnet: one-hour consultation with a crypto-friendly gaming lawyer. Confirm skill-based pari-mutuel classification in your target jurisdiction. Low risk at this stage but worth having on record.
Milestone 4: legal review complete. Green light for mainnet deployment.
Milestone 5: Locker live on mainnet with real crypto pools. Gambit is a real product.
4
Ruse — second game
Week 4-5
Session 6 — Ruse build
3-4 hrs
Signal game mechanic on top of Locker infrastructure. New game logic module, new player and spectator UI skins, same lobby, same pool contracts, same prize pot logic. Fastest build in the portfolio.
Deliverable: two games live on Gambit
Milestone 6: Gambit has a lobby with two distinct games. Platform feel established.
